🫧 氣泡排序法 (Bubble Sort)

什麼是氣泡排序法？

氣泡排序法是一種簡單的排序演算法。它會重複地走訪要排序的數列，一次比較兩個相鄰的元素。如果它們的順序錯誤（例如：前面的數字大於後面的數字），就把它們交換過來。

這個過程會不斷重複，直到沒有任何一對數字需要交換為止。因為越大的元素會經由交換慢慢「浮」到數列的頂端（最後面），所以被稱為「氣泡」排序。

準備就緒

JavaScript 程式碼範例

function bubbleSort(arr) {
    let n = arr.length;
    // 外層迴圈控制回合數
    for (let i = 0; i < n - 1; i++) {
        let swapped = false;
        // 內層迴圈進行相鄰元素比較
        for (let j = 0; j < n - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                // 交換元素
                let temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                swapped = true;
            }
        }
        // 如果這一回合完全沒有發生交換，代表已經排序完成，可提早結束
        if (!swapped) break;
    }
    return arr;
}
