## 一、 前言
在計算機科學領域中，資料排序（Sorting）是一項不可或缺的核心技術，廣泛應用於資料處理、搜尋最佳化及各類進階演算法的底層架構中。本報告聚焦於探討數種經典的排序演算法，包含：氣泡排序（Bubble Sort）、選擇排序（Selection Sort）、插入排序（Insertion Sort）、快速排序（Quick Sort）以及合併排序（Merge Sort）。我們將從時間複雜度（Time Complexity）與空間複雜度的角度出發，結合理論推導與實際程式測試，深入比較各演算法在不同資料規模與情境下的運算效率與優劣勢。
